About AI Story Adventure

Stories that remember
every choice you make.

AI Story Adventure is an interactive storytelling platform where players create worlds, shape characters, and continue living stories powered by AI.

The Idea

A living story engine.

This project is built around one idea: stories should not be fixed. The world should react, remember, and evolve based on what the player does. Instead of hard-coded scenes, AI generates worlds, characters, conflicts, narrative turns, and meaningful choices dynamically.

Mode One

Adventure Mode

A fast-paced AI text adventure where players act directly inside the world. Every turn is driven by action, exploration, risk, and immediate choices.

  • Dynamic world generation
  • Choice-based gameplay
  • Action-focused turns
  • Save and continue anytime
Mode Two

Novel Mode

A long-form interactive novel experience where players co-write cinematic prose with AI. The system builds a story bible, then writes scenes while still offering choices.

  • World seed or AI-generated world
  • Question wizard setup
  • Long-form prose generation
  • Narrative direction choices

Memory Architecture

The story does not forget.

The app combines raw messages, summaries, important facts, and vector memory retrieval so the AI can keep continuity across long sessions.

✦ Story Summary

Compresses long sessions into stable narrative memory.

◈ Important Facts

Keeps rules, promises, relationships, items, and major events.

▰ Vector Memories

Finds relevant past moments for the current scene.

⌁ Persistent Saves

Stores each user’s sessions for continuing later.

Project Stack

Built as a full-stack AI story project.

Frontend

Game portal UI, story reader, setup wizards, saves, and responsive interaction flow.

Backend

FastAPI routes for sessions, turns, novel setup, memory, and AI provider abstraction.

AI Providers

Gemini, OpenAI, Groq, Ollama, and mock providers through a shared interface.

Persistence

Firebase Auth, Firestore sessions, summaries, facts, and save/load behavior.
